# MT5 Risk Panel

Professional MT5 Expert Advisor for entering positions using line-based entry, automatic risk calculation and  order management.

## Overview

MT5 Risk Panel is a semi-manual EA designed to:

- Calculate position size automatically based on risk percentage
- Use chart-based entry and stop-loss lines
- Enter trades with broker-safe lot normalization
- Provide a clean and modular architecture for expansion

## Core Features (Phase 1)

- Line-based entry and stop-loss placement
- Automatic lot size calculation
- Tick-value-based risk computation (broker independent)
- Symbol-aware volume normalization
- Event-driven chart interaction
